DFC 2026 — Multi-Label Product Ranking

Решение задачи multi-label классификации в рамках соревнования Data Fusion Contest 2026 (Задача 2) от ODS.AI.

📌 Описание задачи

Необходимо предсказать вероятности открытия 41 финансового продукта для клиентов банка на основе обезличенных данных.

Данные:

1 000 000 клиентов

750 000 — обучающая выборка

250 000 — тестовая выборка

~200 основных признаков

2000 дополнительных признаков

Много пропусков и выбросов

Тип задачи:

Multi-label классификация (41 бинарная цель).

Метрика:

Macro Averaged ROC-AUC
Особенности решения

Обработка большого количества пропусков

Работа с выбросами

Multi-label стратегия (one-vs-rest)

Кросс-валидация

Валидация по Macro ROC-AUC

Поддержка обучения на GPU

Автоматическая генерация submission-файла

Используемые модели

CatBoostClassifier (GPU)

One-vs-Rest схема

Эксперименты с различными наборами признаков
